# DroughtWatch Backend API

Stateless Express + TypeScript API with one endpoint:

- `POST /api/alert`

## Setup

1. Copy `.env.example` to `.env` and set:
   - `NASA_EARTHDATA_TOKEN`
   - `GEMINI_API_KEY`
   - `PORT`
2. Install dependencies:
   - `npm install`
3. Run in dev:
   - `npm run dev`
4. Build:
   - `npm run build`
